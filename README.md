# Trust Wallet - Chrome Extension

- Design Pattern: Page Object Model
- Tool: Selenium
- Framework: TestNG
- Reporting: Extent Report
- Parameterised Tests
- Retry on Failure
- Screenshot on Failure

# How to run?

- Right click on the testng.xml and click on the run button.
